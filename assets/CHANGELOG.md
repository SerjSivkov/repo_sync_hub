# Changelog

## 1.3.0

Локализация интерфейса: русский и английский.

- Feat: переключатель языка (RU/EN) — кнопка в шапке и выбор в настройках;
  язык применяется мгновенно и сохраняется между запусками
- Feat: все строки интерфейса, статусы операций и логи вынесены в ARB
  (`lib/l10n/`) с генерацией `AppLocalizations`
- Feat: локализованные относительные даты и единицы размера (КБ/МБ ↔ KB/MB)
  через ICU-плюрализацию
- Test: тесты на настройку языка (значение по умолчанию, round-trip, фолбэк)

## 1.2.0

Быстрый переход на страницу репозитория в браузере.

- Feat: кнопка «Открыть origin в браузере» в строке каждого репозитория
  (и дублирующий пункт в контекстном меню)
- Feat: нормализация remote-ссылки в веб-адрес — поддержка https, scp-подобных
  `git@host:group/repo.git`, `ssh://` и `git://`; токен из URL отбрасывается
- Feat: кнопка неактивна, если у репозитория нет origin-ссылки
- Test: тесты нормализации remote-ссылок для всех форматов

## 1.1.0

Параллельное сканирование и настройка конкуррентности.

- Feat: параллельный опрос репозиториев (пул с ограничением параллелизма) —
  скан десятков репозиториев с `fetch`/`du` стал заметно быстрее
- Feat: настройка «Параллелизм скана» (1–16, по умолчанию 4) в настройках сканирования
- Feat: детерминированный порядок репозиториев в прогрессе и результате — по пути,
  а не по завершению скана
- Feat: отмена скана срабатывает сразу для ещё не стартовавших задач
- Fix: правки ошибки при добавлении директории для сканирования
- Test: добавлены тесты на пул конкуррентности и round-trip новой настройки

## 1.0.0

Первый релиз Repo Sync Hub — macOS-инструмент для массового обслуживания
git-репозиториев: сканирование, pull main/master, push в любую систему-приёмник.

- Feat: сканирование нескольких директорий сразу (список корней в настройках)
- Feat: группировка репозиториев по дочерней директории (1 уровень ниже корня)
- Feat: два вида отображения — построчно и деревом; сворачивание группы по клику на шапку
- Feat: репозитории с ошибками всегда сверху отдельной группой
- Feat: сортировка внутри группы по последним стянутым изменениям; репозитории без активности больше года помечаются как «заброшен»
- Feat: столбец «последний pull», количество коммитов и размер репозитория
- Feat: кэш сканирования — данные хранятся между запусками, свежие репозитории не пересканируются (настраиваемый TTL, дата последнего скана)
- Feat: сканирование по расписанию (включение + интервал в минутах)
- Feat: тёмная тема (система / светлая / тёмная), переключатель в шапке и настройках
- Feat: ярлык на репозиторий — симлинк на Рабочем столе, «Показать в Finder», «Открыть в Терминале»
- Feat: не привязано к GitLab — любая git-система-приёмник (хост / группа / токен / имя remote)
- Feat: подсказки (tooltip) у каждой кнопки и столбца-метрики
- Feat: скрипты релиза для macOS — сборка `.app`, упаковка в DMG + zip, интерактивный `release.dart`
